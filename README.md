# RAG_LLM_Evaluation

Project is part of the Udemy course: RAG LLM Evaluation AI Test.

Focuses on evaluating various aspects of Language Learning Models (LLM based Applications) using different metrics  

## Steps
1. create virtual env and install requirements.txt
2. create nltk_data folder  and subfolder
    - create taggers folder- download and copy Averaged Perceptron Tagger 
    - tokenizers - download and copy punkt_tab folder 
    

3. To run specific test, run using pytest -s -k "{test_name}"


### Environment-variables
1. Create a .env file and add the following environment variables
    - OPENAI_API_KEY=openai_api_key
    - RAGAS_API_TOKEN=ragas_api_key
