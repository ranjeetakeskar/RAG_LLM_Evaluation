# RAG_LLM_Evaluation

Project is part of the Udemy course: RAG LLM Evaluation AI Test.
Focuses on evaluating various aspects of Language Learning Models (LLM based Applications) using different metrics  


## Environment-variables
Create a .env file and add the following environment variables:
    OPENAI_API_KEY=your_openai_api_key
    RAGAS_API_TOKEN= your_ragas_api_key
